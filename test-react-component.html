Component Code Format

The component code must:

  • Be plain JavaScript (no JSX)
  • Use React.createElement() calls
  • Define a component named `ReactComponent`
  • Props are passed via the ["props", "json_string"] tag

Example Component Code

// Simple counter component
const ReactComponent = function(props) {
  const [count, setCount] = React.useState(0);
  
  return React.createElement(
    'div',
    { style: { padding: '10px', border: '1px solid #ddd' } },
    React.createElement('h3', null, props.title || 'Counter'),
    React.createElement('p', null, 'Count: ' + count),
    React.createElement(
      'button',
      { onClick: () => setCount(count + 1) },
      'Increment'
    )
  );
};
