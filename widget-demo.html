This demo shows how to embed the TENEX widget on any website. The widget appears as a floating button in the bottom-right corner.

How to Embed

Add this single line of code to your HTML before the closing </body> tag:

<script
  src="https://tenex.app/widget.js"
  data-project-id="naddr1..."
  defer
></script>

Replace naddr1... with your actual TENEX project ID.
