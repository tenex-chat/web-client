Test Instructions for Multi-Voice Selection

Steps to Test:

  1. Open the app at http://localhost:3002
  2. Navigate to Settings (gear icon)
  3. Go to the "Voice Settings" section
  4. Enable "Text-to-Speech" toggle
  5. Choose your voice provider (OpenAI or ElevenLabs)
  6. Enter your API key for the selected provider
  7. Click the "Select Multiple Voices" button
  8. In the dialog that opens:
    • You should see CHECKBOXES (not radio buttons)
    • You should be able to select multiple voices
    • The dialog title should say "Select Voices" (plural)
    • There should be a count of selected voices
    • The confirm button should say "Select X Voices"
  9. Select 2-3 voices and click the confirm button
  10. After selection, you should see all selected voices listed with X buttons to remove them
  11. The UI should show "Selected Voices (X)" where X is the number of voices

Expected Behavior:

  • ✅ Multiple voice selection using checkboxes
  • ✅ Visual indication of selected voices count
  • ✅ Ability to remove individual voices from selection
  • ✅ Deterministic voice assignment message shown
  • ✅ "Add Voices" button to add more voices to existing selection

Known Working Features:

  • Single voice selection via "Select Voice" button
  • Multi-voice selection via "Select Multiple Voices" button
  • Voice preview functionality
  • Switching between single and multi-voice modes
